  • 齐次线性方程组

    • ==基础解系==：如果 η1,η2,...,ηt\eta_1,\eta_2,...,\eta_tη1​,η2​,...,ηt​ 是齐次方程组 Ax=0Ax=0Ax=0 的解，而且满足

      • η1,η2,...,ηt\eta_1,\eta_2,...,\eta_tη1​,η2​,...,ηt​ 线性无关
      • Ax=0Ax=0Ax=0 的任一个解 η\etaη 都可由 η1,η2,...,ηt\eta_1,\eta_2,...,\eta_tη1​,η2​,...,ηt​ 线性表出

      则称 η1,η2,...,ηt\eta_1,\eta_2,...,\eta_tη1​,η2​,...,ηt​ 是 Ax=0Ax=0Ax=0 的一个基础解系

    • 齐次方程组 Am×nx=0A_{m×n}x=0Am×n​x=0 有非零解 ⇔\Leftrightarrow⇔ r(A)<nr(A)<nr(A)<n （列数）

      • 当 m<nm<nm<n 时，Ax=0Ax=0Ax=0 必有非零解
      • 当 m=nm=nm=n 时，Ax=0Ax=0Ax=0 有非零解 ⇔\Leftrightarrow⇔ ∣A∣=0|A|=0∣A∣=0

    • 基础解系：n−rn-rn−r 个解向量

  • ==非齐次方程组== Ax=bAx=bAx=b （bbb 是非零向量）

    • 非齐次方程组有解 ⇔\Leftrightarrow⇔ r(A)=r(A‾)r(A)=r(\overline{A})r(A)=r(A) （系数矩阵的秩和增广矩阵的秩相等）⇔\Leftrightarrow⇔ bbb 可由 AAA 的列向量线性表出
    • 非齐次方程组有解 ⇔\Leftrightarrow⇔ r(A)+1=r(A‾)r(A)+1=r(\overline{A})r(A)+1=r(A)

  • ==初等变换==：设 AAA 是 m×nm×nm×n 矩阵

    • 用某个非零常数 kkk（k≠0k\not=0k​=0）乘 AAA 的某行（列）的每个元素 （倍乘）
    • 互换 AAA 的某两行（列）的位置 （互换）
    • 将 AAA 的某行（列）元素的 kkk 倍加到另一行（列） （倍加）

  • 初等矩阵与初等变换的性质

    • 初等矩阵的转置仍是初等矩阵
    • 初等矩阵均是可逆阵，且其逆矩阵仍是初等矩阵
      • E(i,j)−1=E(i,j)E(i,j)^{-1}=E(i,j)E(i,j)−1=E(i,j)，E(i(k))−1=E(i(1k))E(i(k))^{-1}=E(i(\frac{1}{k}))E(i(k))−1=E(i(k1​))，E(ij(k))−1=E(ij(−k))E(ij(k))^{-1}=E(ij(-k))E(ij(k))−1=E(ij(−k))
    • 用初等矩阵 PPP 左乘（右乘）AAA，其结果 PA(AP)PA(AP)PA(AP) ，相当于对 AAA 作相应的初等行（列）变换

  • ==等价矩阵==

    • 矩阵 AAA 经过有限次初等变换变成矩阵 BBB，则称 AAA 与 BBB 等价，记成 A≅BA\cong BA≅B
    • 若 A≅(ErOOO)A \cong \left( \begin{matrix} E_r & O\\ O & O\end{matrix} \right)A≅(Er​O​OO​)，则后者称为 AAA 的等价标准形

  • 行阶梯矩阵、行最简矩阵

    • ==行阶梯矩阵==
      • 如果矩阵中有零行（即这一行元素全是0），则零行在矩阵的底部
      • 每个非零元的主元（即该行最左边的第1个非零元），它们的列指标随着行指标的递增而严格增大
    • ==行最简矩阵==
      • 非零行的主元都是1，且主元所在的列的其他元素都是0，则称其为行最简矩阵

  • 线性相关/线性无关

    • ==线性相关==

      • 向量组中的至少一个向量可以由其他向量线性表出
      • 对 mmm 个 nnn 维向量 α1,α2,...,αm\alpha_1,\alpha_2,...,\alpha_mα1​,α2​,...,αm​，若存在不全为零的数 k1,k2,...,kmk_1,k_2,...,k_mk1​,k2​,...,km​，使得 k1α1+k2α2+...+kmαm=0k_1\alpha_1+k_2\alpha_2+...+k_m\alpha_m=0k1​α1​+k2​α2​+...+km​αm​=0 成立，则称该向量组线性相关
      • 以 αj\alpha_jαj​ 为列向量的齐次线性方程组 x1α1+x2α2+...+xmαm=0x_1\alpha_1+x_2\alpha_2+...+x_m\alpha_m=0x1​α1​+x2​α2​+...+xm​αm​=0 有非零解（r(A)<nr(A)<nr(A)<n）

    • ==线性无关==

      • 当且仅当系数 k1=k2=...=km=0k_1=k_2=...=k_m=0k1​=k2​=...=km​=0 时才有 k1α1+k2α2+...+kmαm=0k_1\alpha_1+k_2\alpha_2+...+k_m\alpha_m=0k1​α1​+k2​α2​+...+km​αm​=0

    • 向量 β\betaβ 可由 α1,α2,...,αm\alpha_1,\alpha_2,...,\alpha_mα1​,α2​,...,αm​ 线性表出

      ⇔\Leftrightarrow⇔ 存在实数 k1,k2,...,kmk_1,k_2,...,k_mk1​,k2​,...,km​ 使 k1α1+k2α2+...+kmαm=βk_1\alpha_1+k_2\alpha_2+...+k_m\alpha_m=\betak1​α1​+k2​α2​+...+km​αm​=β

      ⇔\Leftrightarrow⇔ 方程组 [α1,α2,...,αm][x1x2...xm]=β[\alpha_1,\alpha_2,...,\alpha_m]\left[ \begin{matrix} x_1 \\ x_2\\ ... \\ x_m\end{matrix} \right]=\beta[α1​,α2​,...,αm​]⎣⎢⎢⎢⎡​x1​x2​...xm​​⎦⎥⎥⎥⎤​=β 有解

      ⇔\Leftrightarrow⇔ 秩 r(α1,α2,...,αm)=r(α1,α2,...,αm,β)r(\alpha_1,\alpha_2,...,\alpha_m)=r(\alpha_1,\alpha_2,...,\alpha_m,\beta)r(α1​,α2​,...,αm​)=r(α1​,α2​,...,αm​,β)

  • ==极大线性无关组==

    • 向量组 αi1,αi2,...,αir\alpha_{i1},\alpha_{i2},...,\alpha_{ir}αi1​,αi2​,...,αir​（1≤ir≤s1\leq i_r\leq s1≤ir​≤s）是向量组 α1,α2,...,αs\alpha_1,\alpha_2,...,\alpha_sα1​,α2​,...,αs​ 的部分组，满足条件
      • αi1,αi2,...,αir\alpha_{i1},\alpha_{i2},...,\alpha_{ir}αi1​,αi2​,...,αir​ 线性无关
      • 在 αi1,αi2,...,αir\alpha_{i1},\alpha_{i2},...,\alpha_{ir}αi1​,αi2​,...,αir​ 中加入任一向量 αi\alpha_iαi​，则向量组 αi1,αi2,...,αir,αi\alpha_{i1},\alpha_{i2},...,\alpha_{ir},\alpha_iαi1​,αi2​,...,αir​,αi​ 线性相关，则称 αi1,αi2,...,αir\alpha_{i1},\alpha_{i2},...,\alpha_{ir}αi1​,αi2​,...,αir​ 是向量组 α1,α2,...,αs\alpha_1,\alpha_2,...,\alpha_sα1​,α2​,...,αs​ 的极大线性无关组

  • ==秩==

    • 向量组的极大线性无关组的向量个数

    • 矩阵 AAA 中非零子式的最高阶数是 rrr（存在 rrr 阶子式不等于零，rrr 阶以上子式均等于零）

  • 正交矩阵

    • 设 AAA 为 nnn 阶矩阵，若 AAT=ATA=EAA^T=A^TA=EAAT=ATA=E，则称 AAA 为正交矩阵
    • 如 AAA 是正交矩阵 ⇔\Leftrightarrow⇔ AT=A−1A^T=A^{-1}AT=A−1 ⇔\Leftrightarrow⇔ AAA 的行向量（或列向量）都是单位向量且两两正交

  • ==线性空间的同构==

    • 设 VVV 与 V′V'V′ 都是域 FFF 上的线性空间，如果存在 VVV 到 V′V'V′ 的一个双射 σ\sigmaσ ，并且 σ\sigmaσ 保持加法和数乘封闭，即

      σ(α+β)=σ(α)+σ(β)σ(kα)=kσ(α)\sigma(\alpha+\beta)=\sigma(\alpha)+\sigma(\beta)\\ \sigma(k\alpha)=k\sigma(\alpha) σ(α+β)=σ(α)+σ(β)σ(kα)=kσ(α)

      则称 σ\sigmaσ 是 VVV 到 V′V'V′ 的同构映射（简称为同构），此时称 VVV 与 V′V'V′ 是同构的，记作 V≅V′V\cong V'V≅V′。

    • 即如果两个线性空间都属于同个域，并且这两个线性空间之间存在一个同构映射，则这两个线性空间是同构的

    • 性质：https://zhuanlan.zhihu.com/p/536146120

      • 同构映射的逆映射以及两个同构映射的乘积还是同构映射
      • 域 FFF 上两个有限维线性空间同构的充分必要条件是它们的维数相等
